# RUC-Thesis-Template (中国人民大学本科毕业论文 LaTeX 模板)

本项目为中国人民大学本科生毕业论文（设计）提供了一个简洁、高效的 LaTeX 模板。基于原作者的优秀工作，本版本对项目结构进行了模块化重构，将繁杂的格式控制与核心的论文撰写完全分离，旨在让学术写作的过程更加纯粹。

## 优化与重构内容

相较于原始的单文件版本，本项目进行了以下优化：

1. **结构模块化**：将所有页面设置、宏包调用、字体声明等格式控制代码分离至独立的 `ructhesis.sty` 宏包文件中，极大精简了主文档 `manuscript.tex` 的代码量。
2. **内容与格式分离**：写作者现在只需在主文档中专注于学术逻辑的编排和正文撰写，无需在海量排版代码中寻找正文入口。
3. **精准保留原版设定**：严格保留了原模板的所有视觉格式，未对最终生成的 PDF 视觉效果做任何改变。
4. **编译引擎适配**：针对最新的 LaTeX 环境进行了微调，确保在 XeLaTeX (2025) 引擎下能够稳定、无警告地编译。

## 分步使用说明

为了顺利编译生成论文，请按以下步骤操作：

**第一步：检查运行环境与编译器**
- 确保你使用的是完整版的 TeX Live 或 MacTeX。
- 本模板**必须使用 XeLaTeX 引擎**进行编译（依赖 `ctexart` 及系统本地中文字体）。
- 参考文献管理使用 `Biber` 引擎。

**第二步：准备项目依赖文件**
请确保你的项目根目录下包含以下核心文件，缺失任何一个都可能导致编译失败：
- `manuscript.tex`：论文主文档（在此处撰写你的论文内容）。
- `ructhesis.sty`：格式宏包文件（无需随意修改）。
- `references.bib`：参考文献数据库文件。
- `head.png`：人大的校徽图片（用于页眉生成）。

**第三步：执行编译流程**
推荐的标准化编译顺序如下，以确保交叉引用和目录正确生成：
1. `xelatex main.tex`
2. `biber main`
3. `xelatex main.tex`
4. `xelatex main.tex`

*(注：如果你使用的是 Overleaf 或配置完善的 VS Code / TeXstudio 环境，通常可以直接点击“编译”按钮，编辑器会自动执行上述完整流程。)*

## 注意事项

- **文件编码**：所有 `.tex`, `.sty` 及 `.bib` 文件必须使用 **UTF-8** 编码保存，否则会出现中文乱码。
- **格式冲突**：如遇学校官方 Word 模板要求与本指导手册冲突，一切以教务处最新发布的《指导手册》为准。
